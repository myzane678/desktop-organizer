# 桌面整理

Windows 桌面整理工具，用于扫描桌面文件、快捷方式和文件夹，按用途自动分类，并在确认预览后重排桌面图标。

## 功能

- 扫描用户桌面和公共桌面项目
- 读取快捷方式目标路径辅助分类
- 按默认规则、手动调整和本地画像进行分类
- 网格、列表、桌面三种视图
- 拖拽或右键修正分类
- 整理前预览桌面布局
- 支持竖向/横向布局预览
- 支持保存布局方案和恢复历史快照
- 支持把选中的图标收纳到文件夹
- 本地保存规则、分类状态、布局快照和用户偏好

## 环境要求

- Windows 10/11
- Node.js
- npm

## 下载安装包

进入 GitHub Releases 页面下载最新版本：

```text
https://github.com/myzane678/desktop-organizer/releases
```

推荐普通用户下载 NSIS 安装包：

```text
桌面整理 Setup 1.0.0.exe
```

安装后会自动创建：

- 桌面快捷方式
- 开始菜单快捷方式
- 卸载入口

如果不想安装，也可以下载 portable 免安装版本，双击运行即可；portable 版本不会自动创建桌面快捷方式。

## 本地运行

```bash
npm install
npm start
```

开发模式：

```bash
npm run dev
```

开发者打包 Windows 安装包：

```bash
npm run dist
```

打包产物会生成到 `dist/` 目录：

- NSIS 安装包：适合普通用户下载安装，会创建桌面和开始菜单快捷方式。
- portable 免安装版：适合临时使用，不会写入安装信息。

分类规则验证：

```bash
npm run verify:classifier
```

## 本地数据

应用会在本机用户目录下创建配置目录：

```text
~/.desktop-organizer/
```

其中保存分类规则、手动分类、分类画像、布局快照、网格校准、隐私同意记录等本地数据。

## 使用前注意

- “确认整理”会移动桌面图标位置。
- “收纳到文件夹”会真实移动被选中的文件或快捷方式。
- 首次使用建议先保存当前桌面快照，并确认预览符合预期后再整理。
- 当前版本主要面向 Windows 桌面环境。

## 发布状态

当前版本支持源码运行和 Windows 安装包构建。GitHub Release 会提供源码包；如果发布者上传 `dist/` 中生成的 `.exe`，普通用户可直接下载安装。
